Måndag 13 januari

Ett mjukt svar stillar raseri, men hårda ord väcker vrede. (Ords. 15:1)

Vad kan vi göra om vi hamnar i en spänd situation, till exempel om någon börjar prata illa om Jehova eller Bibeln? Då behöver vi be Jehova om helig ande och om vishet att kunna svara på ett milt sätt. Men tänk om vi inser efteråt att vi inte reagerade så bra som vi borde ha gjort? I så fall kan vi be om det igen och fundera på vad vi kan göra bättre nästa gång. Då kommer Jehova att ge oss av sin heliga ande så att vi kan behärska oss och visa mildhet. Det finns flera bibelverser som kan hjälpa oss att visa mildhet när vi känner oss frestade att ge svar på tal. Guds ande kan hjälpa oss att komma ihåg sådana verser. (Joh. 14:26) I Ordspråksboken, till exempel, hittar vi flera principer som kan hjälpa oss att vara milda. (Ords. 15:18) Där kan vi också se varför det är så bra att hålla sig lugn i laddade situationer. (Ords. 10:19; 17:27; 21:23; 25:15) w23.09 15 § 6, 7
